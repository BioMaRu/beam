# Beam Biolerplate

Beam is [Hime Go Web Framework](https://github.com/acoshift/hime) Biolerplate.


## how to use

```terminal
go get -u github.com/acoshift/goreload
```

and then run app

```terminal
goreload -x vendor -x assets --all

// if you use makefile just run
make dev
```
